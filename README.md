# Chrome DevTools

MCP server for the Chrome DevTools Protocol. Inspect, debug, and interact with Chrome browser pages.

## Installation

```bash
# Add the marketplace (if not already added)
claude plugin marketplace add wow-look-at-my-code/cc-marketplace#latest

# Install this plugin
claude plugin install chrome-devtools
```

## Features

### MCP Server

- `chrome-devtools` - Provides Chrome DevTools Protocol tools via MCP

## Requirements

- Node.js and npm (for npx)
- Google Chrome or Chromium browser
